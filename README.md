# Bubble UI + Local Video (diving.mp4)

## What this is
A small React app using `react-bubble-ui` that shows a draggable bubble field of icons.
Click any bubble to open a video modal that plays `diving.mp4`.

## Setup
1) Ensure you have Node.js installed.
2) Install dependencies:
   - npm install
3) Put your video file here:
   - public/diving.mp4
4) Run:
   - npm start

## Deploy to GitHub Pages (quick notes)
- Place `diving.mp4` in `public/`.
- Use any GitHub Pages workflow you like (gh-pages, actions, etc.).
